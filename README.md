# 🚧 PotholeStroom

> **Community-driven pothole reporting and repair funding app built with .NET MAUI and Firebase**

PotholeStroom is a cross-platform mobile application (still in development phase as a prototype) that allows road workers to report potholes and enables drivers to contribute towards repair costs. The application keeps track of funding progress and repair status, making the repair process more transparent for everyone involved.

The project was built to gain experience with cross-platform mobile development, cloud databases, user authentication, and real-time data synchronization.

---

## ✨ Features

- 🕳️ Report potholes with location and repair cost
- 👷 Separate Driver and Road Worker user roles
- 💰 Contribute towards pothole repair costs
- 📊 View funding progress for each pothole
- 🔔 Notifications when repairs are fully funded
- ✅ Track potholes from report to repair
- ☁️ Real-time data synchronization using Firebase

## 📖 How It Works

1. Create an account as a Driver or Road Worker.
2. Road Workers report potholes and estimated repair costs.
3. Drivers browse reported potholes.
4. Drivers contribute towards repair costs.
5. Once fully funded, the Road Worker is notified.
6. The pothole is repaired and marked as completed.
